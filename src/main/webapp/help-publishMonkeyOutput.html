Selecting this option will examine the output of the Android Application Exerciser Monkey tool and publish a summary to the build page.
If it is determined that monkey crashed the application or a not-responding situation occurred, the build will be marked as unstable.

Currently, this expects a file called "monkey.txt" to exist in the root of the build workspace.

To run monkey automatically on an Android emulator or device and write the results to monkey.txt, select "Add build step" above and choose "Run Android monkey tester".
